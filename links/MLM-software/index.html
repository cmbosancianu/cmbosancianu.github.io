  1. The lme4 package for R, with its accompanying manual. The package is now actively maintained by Ben Bolker.
  2. The nlme package for R, with its accompanying manual - also allows for non-linear models. Created by Jos√© Pinheiro and Douglas Bates, but now actively maintained by the R-Core team.
  3. Jeroen C. L. Ooms' web-based interface for lme4, hosted by the UCLA servers, which allows for good demo functionality for the package. Jeroen also makes available a similar interface for the ggplot2 package, which is also good fun for creating graphs.
  4. The Centre for Multilevel Modeling at Bristol also maintains MLwiN, another great software for running mixed-effects models - free for UK academics and UK-based students; teaching version is free, paid otherwise, but with a trial license that lasts for 30 days.
  5. HLM, available only for Windows unfortunately, but also capable of handling non-linear mixed-effects models.
  6. Still the standard in terms of ease of learning the syntax for a variety of multilevel models and structural equation models is MPlus - a demo version is available for free. To my knowledge, the only software which can run a three-level multilevel structural equation model with a dichotomous dependent variable (if one is in need of that). Paras Mehta's xxM also promises to have this ability, although I haven't given it a try myself... yet.

SPSS and Stata are natural options as well - the only reason why they haven't been covered in this list is because they're so well known. Stata's xtmixed command does the job beautifully, and SPSS's capabilities in multilevel modeling have increased considerably since being taken over by IBM.
