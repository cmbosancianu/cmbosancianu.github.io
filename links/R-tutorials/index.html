This is a collection of online tutorials that should quickly get you up and running basic procedures in R. They couldn't possibly offer the same depth of coverage as William Venables and Brian Ripley's Modern Applied Statistics with S [link], Michael Crawley's The R Book [link], John Verzani's Using R for Introductory Statistics [link], or Brian Everitt and Thorsten Hothorn's A Handbook of Statistical Analyses using R [link], but they'll do for a beginner that's trying to quickly get the hang of a few basic procedures in R (in fact, a comprehensive list of books on R programming, or on various statistical topics that make use of R for demonstration purposes, can be found here).

  1. Germán Rodríguez at Princeton makes available an online tutorial, which covers basic objects which R uses, reading in data, and running linear models and generalized linear models.
  2. Hadley Wickham (developer of ggplot2, reshape and plyr) offers a tutorial in advanced R programming that is suitable for those who want to get the most out of the flexibility which R offers, compared to other statistical environments. The sections offer some basic information as well: basic operations with R, objects, functions.
  3. Surprisingly, ComputerWorld magazine offers this basic tutorial into descriptive statistics and graphing with R.
  4. Pairach Piboonrungroj maintains a long list of links where tutorials can be accessed.
  5. Vincent Zoonekynd makes available a similar tutorial to that of Germán Rodríguez (see above), although it's far more extensive in scope: graphics, linear regression, Principal Components Analysis, time series, even multilevel models. The information is a bit old (2007), although the basics of R haven't really changed since then.
